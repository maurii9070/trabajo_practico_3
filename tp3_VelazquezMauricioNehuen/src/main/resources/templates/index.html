¿Qué es Java? Guía de Java para principiantes

Java es un lenguaje de programación de propósito general, concurrente, orientado a objetos, que fue diseñado específicamente para tener tan pocas dependencias de implementación como fuera posible.

Java es un lenguaje de programación de alto nivel, lo que significa que es más fácil de leer y escribir que otros lenguajes de programación. También es un lenguaje de programación orientado a objetos, lo que significa que se basa en la idea de que todo en un programa es un objeto.
